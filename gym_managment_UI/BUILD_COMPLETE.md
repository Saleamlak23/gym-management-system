## 🐛 Troubleshooting Quick Links

| Issue | Solution |
|-------|----------|
| Tailwind not working | Ensure `content` paths in `tailwind.config.js` are correct |
| Icons missing | Run `npm install lucide-react` again |
| Build fails | Delete `node_modules` and `package-lock.json`, then `npm install` |
| Port 5173 in use | Run on different port: `npm run dev -- --port 3000` |
| TypeScript errors | Check `tsconfig.json` paths and types |
| Charts not showing | Verify Recharts is installed: `npm install recharts` |
| State not updating | Check Zustand store implementation |
